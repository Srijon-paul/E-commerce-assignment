# E-Commerce Assignment

A modern static e-commerce website built with HTML, CSS, and JavaScript. The project includes responsive UI layouts, product filtering and search, cart management, wishlist support, coupon handling, dark mode, and local storage persistence.

## Project Structure

- `Ecommerce-Website/`
  - `index.html` — Home page
  - `shop.html` — Product listing page
  - `product_details.html` — Product detail page
  - `cart.html` — Shopping cart page
  - `checkout.html` — Checkout page
  - `wishlist.html` — Wishlist page
  - `style.css` — Global styles and responsive layout
  - `js/` — Client-side JavaScript modules
  - `img/` — Project images and assets

## How to Use

1. Open `Ecommerce-Website/index.html` in a browser.
2. Browse products from the shop page.
3. Add items to the cart or wishlist.
4. Apply coupons and manage quantities.
5. Use dark mode and responsive layout on mobile or tablet.

## Key Features

### UI
- Home Page with product highlights and quick navigation
- Full Navbar across pages for easy browsing
- Hero Section to promote featured products
- Footer with site links and contact info
- Responsive layout for desktop, tablet, and mobile

### Product Module
- Product Cards display items with image, name, and price
- Product Data drives the shop and product listings
- Category Filter to show products by category
- Search Functionality for quick product discovery

### Cart Module
- Add to Cart functionality from product listings
- Remove Item support inside the cart page
- Quantity Buttons for incrementing and decrementing items
- Price Calculation with real-time totals
- Checkout Page to review and complete purchases

### Wishlist & Storage
- Wishlist feature to save favorite products
- Coupon Logic to apply discounts to orders
- Save Data in Local Storage for cart, wishlist, coupons, and dark mode
- Dark Mode toggle for light/dark themes
